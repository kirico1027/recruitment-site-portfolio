# コンポーネント一覧（interview / インタビュー一覧）

## 既存コンポーネント（流用）

| コンポーネント名 | 使用箇所 | 備考 |
|---|---|---|
| `top-header` | 全ページ | ブログに `--current` |
| `blog-filters` | カテゴリフィルター | ブログ一覧と同一マークアップ |
| `blog-card` | 記事グリッド | トップ・ブログと同一 |
| `blog__grid` | 記事一覧 | 3列グリッド（3件表示） |
| `section-cta` | ページ下部 | 2カラムCTA |
| `top-footer` | フッター | ダーク背景 |
| `figma-media` | 記事画像 | `--blog-listing` |

## ページ固有

| 要素 | 理由 |
|---|---|
| `blog-interview-intro` | パンくず + ページ見出し（リード文なし） |
| `blog--interview-listing` | インタビュー一覧のセクション余白・グリッド調整 |

## 新規作成するコンポーネント

なし（ブログ一覧・ニュース一覧・社員紹介一覧の既存パーツの組み合わせで構成）
