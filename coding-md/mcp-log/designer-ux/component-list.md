# コンポーネント一覧（designer-ux / UXデザイナー募集詳細）

## 既存コンポーネント（流用）

| コンポーネント名 | 使用箇所 | 備考 |
|---|---|---|
| `top-header` | 全ページ共通ヘッダー | 募集職種をカレント表示 |
| `top-cta` | ページ下部CTA | 既存2カラム構成を流用 |
| `top-footer` | フッター | 共通のダークフッター |

## ページ固有

| 要素 | 理由 |
|---|---|
| `designer-ux-intro` | 4階層パンくず + ページ見出し |
| `designer-ux-body` | 2カラムレイアウト（本文 + サイドバー） |
| `designer-ux-content` | 募集詳細テキスト（見出し・リスト・段落） |
| `designer-ux-entry` | 右カラムのエントリーサイドバー |
