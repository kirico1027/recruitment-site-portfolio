# コンポーネント一覧（news / ニュース一覧）

## 既存コンポーネント（流用）

| コンポーネント名 | 使用箇所 | 備考 |
|---|---|---|
| `top-header` | 全ページ | ブログに `--current` |
| `blog-filters` | カテゴリフィルター | ブログ一覧と同一マークアップ |
| `blog-card` | 記事グリッド | トップ・ブログと同一 |
| `blog__grid` | 記事一覧 | 3列グリッド（2件表示） |
| `section-cta` | ページ下部 | 2カラムCTA |
| `top-footer` | フッター | ダーク背景 |
| `figma-media` | 記事画像 | `--blog-listing` |

## ページ固有

| 要素 | 理由 |
|---|---|
| `news-intro` | パンくず + ページ見出し（リード文なし） |
| `blog--news-listing` | ニュース一覧のセクション余白・グリッド調整 |

## 新規作成するコンポーネント

なし（ブログ一覧・トップ実装済みパーツの組み合わせで構成）
