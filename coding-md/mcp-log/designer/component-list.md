# コンポーネント一覧（designer / デザイナー職種詳細）

## 既存コンポーネント（流用）

| コンポーネント名 | 使用箇所 | 備考 |
|---|---|---|
| `top-header` | 全ページ共通ヘッダー | 募集職種をカレント表示 |
| `top-cta` | ページ下部CTA | 既存2カラム構成を流用 |
| `top-footer` | フッター | 共通のダークフッター |
| `blog-card` | デザイナーのブログ一覧 | タイトル/日付カードを流用 |
| `figma-media` | ヒーロー画像・ブログ画像 | 既存画像表示コンポーネントを使用 |

## ページ固有

| 要素 | 理由 |
|---|---|
| `designer-intro` | パンくず付きのページ見出し専用レイアウト |
| `designer-hero` | 下層専用の横長キービジュアル |
| `designer-skill` | 番号付きスキル解説2カラム |
| `designer-description` | 複数段落の職種説明本文 |
| `designer-positions` | デザイナー募集ポジション一覧 |
| `designer-blog` | デザイナー関連記事のセクション文脈調整 |
