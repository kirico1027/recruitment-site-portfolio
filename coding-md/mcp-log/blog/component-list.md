# コンポーネント一覧（blog / ブログ一覧）

## 既存コンポーネント（流用）

| コンポーネント名 | 使用箇所 | 備考 |
|---|---|---|
| `top-header` | 全ページ | ブログに `--current` |
| `blog-card` | 記事グリッド | トップの社員ブログと同一マークアップ・スタイル |
| `section-heading` | 最新の記事見出し | トップと同一 |
| `top-cta` | ページ下部 | 2カラムCTA |
| `top-footer` | フッター | ダーク背景 |
| `figma-media` | 記事画像 | `--blog` / `--blog-featured` |

## ページ固有

| 要素 | 理由 |
|---|---|
| `blog-intro` | 下層ページ専用の見出し＋リード（2カラム） |
| `blog-featured` | 注目記事（大画像＋日付・カテゴリ・タイトル） |
| `blog-filters` | カテゴリフィルター（ニュース・社員紹介・インタビュー） |
| `blog--listing` | 9件グリッド・セクション余白の文脈調整 |

## 新規作成するコンポーネント

なし（トップページ実装済みパーツの組み合わせで構成）
